Smart garbage collection with bonus card system

Role: Developer 3 Months

In this project, we designed an innovative garbage collection system incorporating a bonus point mechanism using RFID technology. This system encourages individuals to segregate waste by type, streamlining the waste management process and facilitating efficient recycling. By promoting proper waste separation, the solution simplifies processing and creates opportunities to sell recyclable materials to vendors, fostering a more sustainable and eco-friendly approach to waste management.

Objective: The goal of the project is to design a new garbage collection system equipped with an RFID-based bonus point system to encourage waste segregation by type, simplifying waste processing and enhancing its value for recyclers.
